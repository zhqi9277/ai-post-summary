为 WordPress 文章生成和管理 AI 摘要、标签与关键词，支持小工具、正文自动插入、管理分页、批量任务与日志。

== Description ==

AI Post Summary 是一个面向 WordPress 文章页的 AI 内容增强插件。

主要功能：

* 为文章生成 AI 摘要
* 为文章补全标签与关键词
* 支持文章页小工具显示
* 支持正文顶部自动插入
* 支持摘要管理、手动编辑与重新生成
* 支持标签/关键词管理、手动编辑与重新生成
* 支持历史文章批量补摘要、补标签与关键词
* 支持日志查看与按级别筛选

== Installation ==

1. 将 `ai-post-summary` 上传到 `/wp-content/plugins/` 目录
2. 在 WordPress 后台启用插件
3. 进入 `设置 -> AI 摘要` 配置接口地址、API Key、模型和生成规则
4. 如需在文章页展示，可前往 `外观 -> 小工具` 添加 `AI 文章摘要`

== Frequently Asked Questions ==

= 支持哪些 AI 接口？ =

支持兼容 Chat Completions 请求格式的接口。

= 可以只在文章页显示吗？ =

可以。插件提供文章页小工具，也支持正文自动插入。

= 是否支持历史文章补摘要？ =

支持。可在 `设置 -> AI 摘要 -> 任务` 中批量排队。

= 是否支持标签与关键词自动生成？ =

支持。插件可为缺少标签或关键词的文章自动补全，也支持后台手动管理。
